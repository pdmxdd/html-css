HTML is also our container. It must include any CSS, or JavaScript we want to include with this project. You can write inline JavaScript with a script tag, or anline CSS by defining the style of an HTML tag.

However a better solution to keep your HTML, CSS, and JavaScript in separate, easy to maintain files is to use external CSS, and JavaScript. You would need to create the files separately and link to them in the head tag of your HTML file.
